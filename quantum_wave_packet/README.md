# Численное моделирование эволюции волнового пакета квантовой частицы

## Описание проекта

Данный проект посвящён численному моделированию динамики квантовой частицы в одномерных потенциальных полях на основе нестационарного уравнения Шрёдингера.  
Целью проекта является наглядная демонстрация фундаментальных квантовых эффектов, таких как дисперсия волнового пакета, отражение и туннелирование через потенциальный барьер, локализация в потенциальной яме и колебания в гармоническом осцилляторе.

Проект реализован на языке Python и оформлен в виде модульного Python-пакета, что обеспечивает корректную работу импортов и масштабируемость кода.

---

## Математическая модель

В основе моделирования лежит одномерное нестационарное уравнение Шрёдингера:

\[
i\hbar \frac{\partial \psi(x,t)}{\partial t}
=
\left(
-\frac{\hbar^2}{2m}\frac{\partial^2}{\partial x^2}
+ V(x)
\right)\psi(x,t)
\]

где:
- \(\psi(x,t)\) — волновая функция квантовой частицы,
- \(V(x)\) — внешний потенциал,
- \(\hbar\) — приведённая постоянная Планка,
- \(m\) — масса частицы.

В проекте используются безразмерные единицы (\(\hbar = 1\), \(m = 1\)).

Численное решение уравнения выполняется методом Кранка–Николсона, который является неявной и устойчивой по времени схемой.

---

## Реализованные потенциальные модели

В рамках проекта реализованы следующие одномерные потенциалы:

- Свободная частица \(V(x) = 0\)
- Потенциальный барьер конечной высоты
- Конечная потенциальная яма
- Гармонический осциллятор \(V(x) = \frac{1}{2} \omega^2 x^2\)

## Основной сценарий (`main.py`)

Файл `main.py` запускает базовый сценарий моделирования, включающий:
- задание пространственной и временной сетки,
- инициализацию волнового пакета,
- выбор потенциального поля,
- численное решение уравнения Шрёдингера,
- визуализацию плотности вероятности и её временной эволюции.

### Запуск основного сценария

Из корневой директории проекта:

bash
python -m quantum_wave_packet.main

Экспериментальные сценарии

В папке experiments/ реализованы независимые физические эксперименты, каждый из которых демонстрирует отдельный квантовый эффект.

1. Свободная частица (free_particle.py)

Моделируется свободное распространение волнового пакета и его дисперсия во времени.

Запуск:
python -m quantum_wave_packet.experiments.free_particle

2. Квантовое туннелирование (potential_barrier.py)

Исследуется отражение и прохождение волнового пакета через потенциальный барьер конечной высоты.
Вычисляется вероятность туннелирования.

Запуск:
python -m quantum_wave_packet.experiments.potential_barrier

3. Конечная потенциальная яма (finite_well.py)

Демонстрируется локализация квантовой частицы и динамика волнового пакета в потенциальной яме.

Запуск:
python -m quantum_wave_packet.experiments.finite_well

4. Гармонический осциллятор (harmonic_oscillator.py)

Моделируются квантовые колебания волнового пакета в гармоническом потенциале.

Запуск:
python -m quantum_wave_packet.experiments.harmonic_oscillator

Образовательная ценность проекта

Проект объединяет методы квантовой механики, численные алгоритмы и программную реализацию, позволяя глубже понять физический смысл уравнения Шрёдингера и особенности динамики квантовых систем.
